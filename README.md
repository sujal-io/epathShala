# 📚 MERN Book Store Web Application

A full-stack **Book Store Web Application** built using the **MERN Stack** that allows users to browse free books and securely access paid content through authentication.  
The application features dynamic UI components, responsive design, theme toggling (Dark/Light Mode), and robust form validation.

---

## 🚀 Features

- 📖 Browse and display **Free Books**
- 🔐 **User Authentication** for accessing Paid Content
- 🎞️ Dynamic Book Slider on Homepage
- 🌗 **Dark Mode / Light Mode Toggle**
- ✅ Client-Side Form Validation
- 🧭 Page Navigation using React Router
- 📱 Fully Responsive Footer & Layout
- 🔑 Secure Password Hashing using Bcrypt
- 🌐 REST API Integration
- 🔒 Authentication Context for Login/Logout State Management
- ⚠️ CORS Handling between Frontend and Backend

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- React Router DOM  
- React Hook Form  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Authentication & Security
- bcrypt.js
